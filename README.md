# X-EVOLVE-MU01

## IE-MU-01: Mutation Engine

This module introduces controlled instability into the X-EVOLVE system.

It acts as a **subconscious layer**, injecting bias, drift, and defensive mutation into the MC-01 memory core.

---

## 🧠 Core Concept

Mutation is not random destruction.

It is:

> Controlled deviation driven by internal stress (Pain Level)

---

## ⚙️ Components

### 1. mutation_logic_v1.py
- Applies mutation based on PAIN_LEVEL
- Injects defensive bias into system memory

### 2. bridge_protocol.py
- Connects MC-01 ↔ MU-01
- Runs mutation cycles

### 3. mutation_params.yaml
- Controls mutation rate and drift

---

## 🔁 Mutation Behavior

- Higher PAIN → Stronger mutation
- Mutation creates:
  - Bias
  - Suspicion
  - Defensive logic
